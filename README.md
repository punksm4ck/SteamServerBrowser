# Steam Games Server Browser
An enterprise-grade, high-performance GUI for browsing Steam game servers, developed on Kubuntu and optimized for Windows 11.

## Windows Downloads
[![Build Windows EXE](https://github.com/punksm4ck/SteamServerBrowser/actions/workflows/windows-build.yml/badge.svg)](https://github.com/punksm4ck/SteamServerBrowser/actions/workflows/windows-build.yml)

**Latest Release:** [Download SteamServerBrowser_Win.exe](https://github.com/punksm4ck/SteamServerBrowser/actions) (Navigate to the latest green run and scroll to 'Artifacts').

## Features
* **Cross-Platform:** Native support for Linux (Kubuntu) and Windows 11.
* **Network Optimized:** Designed for high-speed fiber connections like your Spectrum 1Gbps plan.
* **Steam Integrated:** Automatically detects local Steam installations via Registry/Home paths.
